# 4D-Miner-AutoBuilder
use python to controll you to build. 
the program simulates mouse and keyboard inputs, it builds the preset outcome

ver0.30版本主要更新：

1.断点重建。在建造中按F8可使程序在安全点停止建造，在下一次运行时可选择继续。若不正常停止程序可能导致建造错误结构。\

2.自动校准有关参数。移动后通过读坐标判断是否成功移动。\

3.建造动作优化，速度更快\

大部分为ai编写的代码，可能有很多bug
